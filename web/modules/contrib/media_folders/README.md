# Media Folders

This module provides a better UI for managing and selecting Media entities in a folder structure.

This folder structure is a Taxonomy vocabulary and is not reflected in the physical location of the files.

The module offers a "windows explorer" style navigation UI, with drag an drop to move things around e drag to upload new files.


For a full description of the module, visit the
[project page](https://www.drupal.org/project/media_folders).

Submit bug reports and feature suggestions, or track changes in the
[issue queue](https://www.drupal.org/project/issues/media_folders).

information, see
[Installing Drupal Modules](https://www.drupal.org/docs/extending-drupal/installing-drupal-modules).


## Features

- Smart file uploads (files are automatically assigned to Media bundles)

- Drag to upload

- Drag and drop to move

- Multiple view modes and sorts

- Search filter inside folders

- Form display widget for easy Media selection

- View display widget for managing multiple bundle presentation

## Configuration

Go to "/admin/config/media-folders"
