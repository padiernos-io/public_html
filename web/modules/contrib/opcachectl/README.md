# OPcache Control

**OPcache Control** (`opcachectl`) is a Drupal module that provides administrators with tools to monitor and manage
PHP's OPcache directly from the Drupal admin interface.

### Key Features:

- **Status Monitoring**: View detailed PHP OPcache statistics and status information, including cache hit rates,
  memory usage, and operational state
- **Configuration Display**: Access and review current OPcache configuration settings and directives
- **Cache Management**: Clear/reset the OPcache when needed through a admin form and a dedicated route to clear
  the cache from external tools.
