## Minifier HTML

Minifier HTML module for Drupal 8+ community users-
Minifier HTML is a performance-focused Drupal 8+ module that optimizes your website by reducing the size of HTML output and compressing inline CSS and JavaScript. Smaller page sizes lead to faster load times, improved user experience, and better SEO scores. This module is ideal for sites that want to enhance performance without modifying their existing code.

Minify HTML and any CSS or JS included in your markup.
Compress your site's HTML and it helps to reduce page size of your rendered output.

## How to use the module

Just install the module no settings are required to compress the HTML.

## Requirements

No additional modules are required.

## Dependencies

No additional modules are required.

Compatible with Drupal 8, 9, and 10.

## Key Features

Minifies HTML output to reduce page size.

Compresses inline CSS and JavaScript for faster page load times.

Helps improve site performance and user experience.

## Architecture

Automatically intercepts Drupal's rendered HTML output before it is sent to the browser.

Minifies HTML by removing unnecessary spaces, line breaks, and HTML comments.

Compresses inline CSS and JavaScript for improved front-end performance.

Designed to be lightweight and dependency-free.
